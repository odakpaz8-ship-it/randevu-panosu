Senkronizasyon kurulmadı.
Nasıl kurulur? (5 dakika, ücretsiz)
1. dash.cloudflare.com/sign-up → Ücretsiz hesap aç
2. Sol menüden Workers & Pages → Create → Create Worker
3. Bir isim ver (örn: randevu) → Deploy
4. Edit Code'a tıkla, kodu tamamen sil ve aşağıdaki kodu yapıştır → Deploy
⏳ Yükleniyor...
5. Workers sayfasına geri dön → Settings → Bindings → Add Binding
   → KV Namespace seç → Variable: RANDEVU_KV
   → KV Namespace açılır menüsünde Create new namespace → isim: randevu → Kaydet
6. Worker URL'yi kopyalayıp yukarıya yapıştır (örn: https://randevu.ad.workers.dev)
7. Kaydet ve Bağlan butonuna bas ✅
